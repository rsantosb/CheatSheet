# CheatSheet Git Cmds
Repositorio con comandos de ayuda: Git

El estado de los archivos del repositorio puede ser:
- tracked: si ya existían

- untracked: nueva creación

---
*Participante:* Rocío Santos

---
## Clonar:
- git add [URL]

## Realizar seguimiento:
- git add [NombreArchivo]

## Ver cambios:
- git status

- git diff:
Muestra líneas añadidas y eliminadas, para archivos no preparados.

- git diff--cached:
Muestra los cambios para archivos preparados.

## Confirmar cambios:
- git commit -m "commit message"

## Enviar a master:
- git push
